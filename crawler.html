Python Web Crawler
Project：
運用 Python 爬蟲方法抓取網頁資料進行分析，並使用 Robo 3T 建立 MongoDB 資料庫。
一、題目：博客來中文書暢銷排行榜爬蟲
二、說明題目：
（一）資料用途：分析排行榜書籍的種類以及普遍價位，藉此看出市場需求
（二）預期結果：爬取博客來中文書暢銷前一百名之書名、作者、售價
三、觀察：博客來排行榜的整體架構很完整，且可以推廣到同網站其他種類的書以及排序方式
（eg. 西洋文學、商業用書；新書上市、依價格排序等）：
四、遇到的問題：
（一）使用 requests 時出現 error 54, 'Connection reset by peer'
解決方法：在 Terminal 輸入 pip install "requests[security]"
（二）網站資料每小時更新一次，導致爬下來的檔案有時候會不一致
解決方法：重頭執行整個python檔，讓全部資料更新至最新
（三）若商品有打折，網站會把價錢跟打折折數放在一起，無法單獨取出價錢
解決方法：抓出「XX折」的 index，再另外寫一個 List，利用 for 迴圈抓出寫在「XX折」後面真正的價錢
（四）價錢超過1000時會加上逗號，而 string 無法判斷、依照數字大小排序
解決方法：將逗號刪除，並把 string 轉成 integer
