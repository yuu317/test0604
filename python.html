Python Game Development
程式目標：
以英打遊戲訓練英打速度及準確性
主程式邏輯：
進入遊戲 → 遊戲開始 → 遊戲結果分析
程式設計：
遊戲：
程式隨機產生一組（共20個）英文字母以及20個氣球，一個氣球裡各放一個字母。計時50秒，玩家需按照氣球順序輸入英文字母，當一組輸入完畢後會隨機跳出下一組。遊戲結束後，分數計分表會出現精確度、速度、分數以及連續答對次數。之後可以選擇繼續遊戲（按Enter）或離開。
視覺設計：
遊戲背景會依照遊戲時間，由白天逐漸轉成黑夜再到白天，共十種亮度，每兩秒變換一次。除了天色改變，還會出現太陽、星星、月亮等。而當分數超過某些特定值時，會出現花草樹木作為獎勵，增加畫面豐富度。顯示英文字母的氣球顏色亦為隨機的。
